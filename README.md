# jsonschematic

jsonschematic is a [json schema][] in the same vein as
[docson][].

## How to run

The project is still in early development, but to get a 'dev'
version of the sapper app running, do:

    $ npm install
    $ JSONSCHEMATIC_DIR=examples npm run dev
